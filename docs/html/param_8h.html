Detailed Description

Configuration management interface.

Deprecated:
This API has been deprecated and replaced by the AXParameter API.
Contains functions for adding, removing, setting and getting parameters.
The interface supports registering callback functions 
when a parameter value is updated.

Example

This code will register a callback, my_callback, to be run when the parameter my_param is updated. It will also add the parameter my_param.

static void list_parameters(void)
{
char **list;
if (param_list(&list) == 0) {
int i;
char *param;
for (i = 0; list[i]; i++) {
if (param_get(list[i], &param) == 0) {
printf("%s=%s\n", list[i], param);
param_free(param);
}
}
param_list_free(list);
}
}
param_stat my_callback(const char *name, const char *value)
{
printf("Parameter '%s' has new value '%s'\n", name, value);
return PARAM_OK;
}
int main()
{
param_init("param_app");
param_register_callback("my_param", my_callback);
if (param_add("my_param", "my_value", "string")) {
return 1;
}
}

Compilation

To build a program using the parameter interface the library name must be supplied as -lparam, the pthreads as -lpthread, and the header file must be included in param.h.
