Prerequisites

Required prior knowledge

Working with the SDK requires:

  • Knowledge of the C programming language.
  • Experience with common development tools such as gcc and make.
  • Experience with Linux® or similar POSIX-compatible platforms.
  • Knowledge about GLib. A good starting point is the GLib Overview and the GLib Reference Manual.
    The GLib version used in the SDK is 2.36 and is the minimum required API version to use when writing application.
    GLib is developed and maintained by the GNOME Project and is released under the GNU Library General Public License (GNU LGPL).

Axis does not provide training or documentation about: the C language, C libraries, GLib, the Linux kernel, developement tools like gcc and make, and similar.

Technical prerequisites

To install and use the SDK, you need:

  • A workstation running a Linux based operation system (Debian or Ubuntu recommended).
  • Bash Unix shell (Bash recommended, other shells may also work).
  • An Axis network video product that supports Embedded Application Development.
    Embedded Application Development is supported if the parameter root.Properties.EmbeddedDevelopment exists.
    To check if the parameter exists, use the following VAPIX® command, for example in a web browser:
    http://camera-ip/axis-cgi/param.cgi?action=list&group=root.Properties.EmbeddedDevelopment
    Verify that the Version parameter is root.Properties.EmbeddedDevelopment.Version=2.00 to support SDK version 2.0 .
  • Cross compilers. The cross compilers are provided by Axis.
    See section Installing the Software Development Kit, or visit Axis Partner Pages.
  • The curl package is required to run the SDK script eap-install.sh. Use your distribution package manager to install the package.
  • The autoconf and libtool packages are required to compile the RAPP library. Use your distribution package manager to install these packages.
  • An Axis product with firmware version 5.60 or greater.
